Loopback Wiring for the NI myRIO

You can use your NI myRIO to both generate a signal and acquire it, by implementing loopback wiring. For example, you can connect Analog Output 0 (AO0) to Analog Input 0 (AI0). To do this, complete the following steps:

1.     Connect a wire from the AGND pin on Connector C to the AI0- pin.

2.     Connect a wire from the AO0 pin on Connector C to the AI0+ pin.

Tutorial: Using the LabVIEW Project

This tutorial explains how to use the Oscilloscope & Function Generator for NI myRIO project. Note: Connect your NI myRIO via USB for this tutorial.

Start the Program

1.     Run RT Main.vi.

2.     Run Desktop Main.vi.

Using the Function Generator

3.     Connect terminals AO0 or AO1 on Connector C to your desired circuit, or loop them back to the myRIO to channels AI0 or AI1. See the Loopback Wiring for the NI myRIO section above for details.

4.     In the Function Generator section, choose your desired Frequency, Amplitude (Vpeak), and Waveform type.

5.     Click Run.

6.     Click Stop to end output generation.

Take a Reading with the Oscilloscope

7.     If you have not already done so, connect a signal to the myRIO AI0 or AI1 terminals on Connector C.

8.     Specify the appropriate input in the Channel 0 Settings or Channel 1 Settings section of Desktop Main.vi.

9.     Click Run in the Acquisition Control section of Desktop Main.vi.

10.  Click Autoscale to autoscale Volts/Div for each channel.

11.  Adjust Time/Div in the Time section so you can visualize the signal more clearly.

12.  Click the Measurements checkbox for the appropriate channelÕs measurements to see Vrms, Vpeak, and Freq measurements.

13.  Click Enable Cursors to show cursors. Drag them to their desired location to see magnitude and time difference.

Create a Function Sweep

14.  Go to the FGen menu and choose Configure.

15.  Click Enable Sweep and specify your desired Start and Stop frequencies, step size, and time between steps (step interval).

16.  Click OK.

17.  In the Function Generator section, specify your desired amplitude (Vpeak) and Waveform type.

18.  Click Run.

End the Program

19.  In Desktop Main.vi, go to the File menu and choose Exit.

20.  In RT Main.vi click Stop.

Tutorial: Saving Oscilloscope Data

This tutorial explains how to save the Oscilloscope data shown in the graph indicator on the front panel.

1.     Acquire data using the Oscilloscope. For more information, see the Tutorial: Using the LabVIEW Project.

2.     Right-click on the Oscilloscope graph.

3.     Choose Export and select any of the available options. If you do not have Microsoft Excel or DIAdem installed, choose Clipboard. Then paste the data into another application.
